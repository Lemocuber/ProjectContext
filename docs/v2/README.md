# V2 Spec Pack

Status: in progress. Milestone 1 completed on 2026-03-12; Milestone 2 completed on 2026-03-13; Milestone 3 completed on 2026-03-13; Milestone 4 is current.

## Scope Pillars
- Recording keepalive across screen lock, app backgrounding, and in-app tab switches.
- Cloud-backed session history and artifact storage for cross-device access.
- In-session "What do you think" suggestions powered by realtime transcript context.
- Privacy-safe remote diagnostics and manual report support for field debugging.
